# python invalid token

在写python代码的时候，import了一个包

```python
import ocp_4.0_utils
```

然后这里报错

```
invalid token
```

意思是导入的包或模块不符合程序命名要求

比如，python的模块包不能以数字开头命名

在这里，`.` 后面跟数字是不允许的
